今天也是不知道熬到多晚才睡觉，估计是五六点吧。还是那个七碗茶劲太大了，已经是少量了，还是睡不着。今天下午一两点才起来，主要是研究我的 remote lab 的项目。

昨天试了一天的 Claw，OpenClaw 已经感觉到这个东西并不是很受控制，并且它不是很方便地能接到 Coding Plan 上。整体来说灵活性不够，而且花钱花的太多。所以我觉得我的方案还是比较适合我的，是我能控制每一个 Agent 在做什么，因为它本质就是调用的 Claude Code 或者 Code X。我还为它增加了整个系统的编排能力和调用子 Agent 的能力。子 Agent 实际上也就是一个 Workspace 里的 Agent，就是一个独立的 Claude Code Folder。那么这样的话，管理性和可控性就极强，而且架构简单，可以直接进去 Debug，看到底哪里出问题了。现在已经增加了可以互相分发任务的功能，今天优化了一些用户体验，继续做下去吧，感觉很有前途，可能可以开源吸引一些人来用。所以如果是这样的话，也不用买什么电脑了，直接把我的那个 aisos ally 作为以后的主要用机，做 Claude Code 就行了，但是它的问题就是不知道怎么把我的 SSH 暴露出去比较安全，毕竟它是在私网的一个东西，是否也要走 Cloudflare 的 tunnel。

然后今天用我的这个 remote lab 的项目做了一个 samurai，也就是直接提取视频的字幕信息来进行总结。毕竟现在这个 AI 太火了，导致现在所有的内容灌水严重，AI 生成的内容，实际上本质上就是通过文字生成的，那我直接通过它的字幕获取它的信息和图片的差异只是不大的，然后获取到信息之后它自己给我评价，我把我现在做的几个项目都告诉他了，它直接分析我的项目，看这个东西是否适合插入我的项目中去，然后如果要插入的话怎么加入，它直接帮我客观分析了之后，就没有我自己去看这个视频的那种情感上的 FOMO 的感觉。有的时候它明确地告诉我这个东西不行，那我也很开心，就是说，我终于不用被这个 FOMO 裹挟了。现在的问题实际上就是 AI 导致的内容爆炸和灌水严重，人人都可以发高质量的视频，就导致所有内容的质量都在下降。所以以毒攻毒，你用 AI 发视频，我用 AI 看视频。谁也别说谁牛逼，谁也别说谁高尚。

今天买了一个打字狂潮的游戏，就是一边练打字一边玩肉鸽游戏的感觉还不错。可能这个能推动我每天练练打字。我以前一直都在找一个能让我手指动起来，还像游戏一样很爽的东西。以前感觉这个应该是师傅 RPG 的打斗游戏，但是感觉上手难度太高了，打不动，就搁置了。今天发现这个游戏确实很适合动手指，你在游戏里有成就感，游戏外也有成就感，因为你确实练打字了，对你自己是有帮助的。感觉挺好。

另外一个游戏买了一个 MineIndustry，这个游戏就很一般了。虽然它的灵活度很高，就是一个拉产线建造加塔防的游戏。但是它现在可能是在一个非常早期的阶段，太简陋了。拉线都不能拐弯拉，你要每次拉一条直线，再做拼接，这样就感觉很不舒服，不流畅。就退款了。

Another night where I have no idea when I finally fell asleep — probably around five or six in the morning. That seven-bowl tea hit way too hard. I already cut back to a small amount and still couldn't sleep. Didn't get up until one or two in the afternoon. Spent most of the day working on my remote lab project.

Yesterday I spent a whole day trying out Claw and OpenClaw. It became pretty clear that the thing isn't very controllable, and it doesn't connect to Coding Plan in any convenient way. Overall not flexible enough, and it burns through money too fast. So I think my own approach still suits me better — I can see exactly what each Agent is doing, because at its core it's just calling Claude Code or Code X. I've also added orchestration capabilities across the whole system, plus the ability to call sub-Agents. A sub-Agent is essentially just an Agent in a Workspace — an independent Claude Code folder. That makes it highly manageable and controllable, the architecture stays simple, and you can go in and debug directly to see what's actually broken. I've now added the ability for agents to distribute tasks to each other. Today I polished some of the user experience and will keep going — it feels promising, might even open-source it and attract some users. If that's the direction, I don't need to buy a new machine. I can just use my aisos ally as my main daily driver and run Claude Code on it. The only problem is I don't know the safest way to expose my SSH, since it's sitting on a private network. Maybe I need to route it through a Cloudflare tunnel.

I also used my remote lab project today to build something I'm calling samurai — it extracts subtitles from videos and summarizes them. The idea is that with AI so hot right now, content everywhere is getting watered down. AI-generated content is essentially text at its core, so grabbing the subtitle track versus watching the actual video isn't that different in terms of information value. It pulls the content, then evaluates it for me. I told it about the projects I'm currently working on, and it analyzed each one to see whether whatever was in the video was actually worth integrating — and if so, how. After it gave me its objective take, that anxious FOMO feeling I'd normally get from watching the video just... wasn't there. Sometimes it tells me flat out: this isn't relevant. And honestly that makes me happy. I'm finally not getting dragged around by FOMO. The real problem right now is that AI has caused a content explosion. Everyone can put out polished-looking videos, which means the actual quality of everything is sliding. So — fighting fire with fire. You use AI to make videos, I'll use AI to watch them. No one gets to claim the moral high ground here.

Bought a game called Typing Mania today — a roguelite where you type to fight. Feels pretty good. Maybe this will actually push me to practice typing every day. I've been looking for something that gets my fingers moving and still feels like a game. Used to think a martial arts RPG combat game would do it, but the learning curve was too steep, so that got shelved. This game, though — it genuinely gets the fingers going. You feel accomplished in the game and accomplished outside it too, because you're actually improving your typing. Real benefits. I like it.

Also bought something called MineIndustry. That one was pretty underwhelming. The flexibility is there — it's a production-line builder with tower defense elements — but it's clearly in very early development. Way too rough. You can't even bend cables; every connection has to be a straight line that you then splice together manually. It just feels awkward and clunky. Refunded it.
